wiced_bt_a2d_status_t wiced_bt_a2d_bld_mdu_info ( uint8_t  media_type,
wiced_bt_a2d_mdu_cie_t *  p_ie,
uint8_t *  p_result 
)

This function is called by an application to build the MPEG-2, 4 AAC Media Codec Capabilities byte sequence beginning from the LOSC octet.

Parameters
[in]media_type: Indicates Audio, or Multimedia.
[in]p_ie: MPEG-2, 4 AAC Codec Information Element information.
[out]p_result: the resulting codec info byte sequence.
Returns
A2D_SUCCESS if function execution succeeded.
Error status code , otherwise.
