wiced_bt_ble_ext_adv_report_t Struct Reference

Description

Extended advertisement report data format.

Note
When multiple advertising packets are used to complete a single advertising report(i.e. if adv set data received as multipl adv reports), the RSSI and TxPower event parameters shall be set based on the last packet received.
