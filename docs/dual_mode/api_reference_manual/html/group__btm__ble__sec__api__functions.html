This API is called by the application to request for notification on change in device address.

Application receives BTM_BLE_DEVICE_ADDRESS_UPDATE_EVENT event in the wiced_bt_management_cback_t which is registered via wiced_bt_stack_init.

Parameters
[in]enable: Enable change of address notification
Returns

WICED_BT_SUCCESS : if Enabled.
WICED_NOT_AVAILABLE : Otherwise.

◆ wiced_bt_ble_read_device_random_address()

wiced_bt_dev_status_t wiced_bt_ble_read_device_random_address ( wiced_bt_device_address_t  random_bd_addr)

Get the configured local random device address.

Note
Random address depends on below settings in that priority order.
1) Global privacy configuration using wiced_bt_cfg_ble_t::rpa_refresh_timeout (see wiced_bt_cfg_settings_t).
2) else, configured for static random bd_address while downloading using BT_DEVICE_ADDRESS=random build setting.
Parameters
[out]random_bd_addr: device random bluetooth address
Returns
wiced_bt_dev_status_t
WICED_BT_SUCCESS : if random address is configured.
WICED_BT_WRONG_MODE : if random address not configured.

◆ wiced_bt_ble_security_grant()

void wiced_bt_ble_security_grant ( wiced_bt_device_address_t  bd_addr,
wiced_bt_dev_status_t  res 
)

API to grant or deny security access.

Used in response to BTM_SECURITY_REQUEST_EVT event. This event is received in the wiced_bt_management_cback_t registered via wiced_bt_stack_init.

Parameters
[in]bd_addr: Bluetooth address of remote device
[in]res: WICED_BT_SUCCESS : To grant access.
WICED_BT_UNSUPPORTED : If local device does not allow pairing.
WICED_BT_REPEATED_ATTEMPTS : Otherwise
Returns
None
