◆ wiced_bt_a2d_bld_m12info()

wiced_bt_a2d_status_t wiced_bt_a2d_bld_m12info ( uint8_t  media_type,
wiced_bt_a2d_m12_cie_t *  p_ie,
uint8_t *  p_result 
)

This function is called by an application to build the MPEG-1, 2 Audio Media Codec Capabilities byte sequence beginning from the LOSC octet.

Parameters
[in]media_type: Indicates Audio, or Multimedia.
[in]p_ie: The MPEG-1, 2 Audio Codec Information Element information.
[out]p_result: the resulting codec info byte sequence.
Returns
A2D_SUCCESS if function execution succeeded.
Error status code otherwise.

This function is called by an application to parse the MPEG-1, 2 Audio Media Codec Capabilities byte sequence beginning from the LOSC octet.

Parameters
[in]p_info: the byte sequence to parse.
[in]for_caps: TRUE, if the byte sequence is for get capabilities response.
[out]p_ie: The MPEG-1, 2 Audio Codec Information Element information.
Returns
A2D_SUCCESS if function execution succeeded.
Error status code otherwise.
