HID Device Role (HIDD) over BLE

General Description

This component maps features from the USB Human Interface Definition onto Bluetooth low energy GATT characteristics and descriptors.

HIDD role defines a set of procedure that can be used by an application like to implement a HID device level functionality over BLE transport.
