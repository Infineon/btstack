HID Device Role (HIDD) over BR/EDR

General Description

HIDD Functions This component maps features from the USB Human Interface Definition onto Bluetooth as a profile. The HID Device (HIDD) role defines a set of procedures that can be used by an application to implement a HID device level functionality over a BR/EDR Transport.
